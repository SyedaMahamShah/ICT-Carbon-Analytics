# ICT-Carbon-Analytics

Lifecycle carbon modeling toolkit for ICT equipment.

## Based on

MSc Thesis:
"Comprehensive Life Cycle Evaluation of Carbon Emission from ICT Equipment"
Author: Maham Noor
Aalto University (2025)

## Features

- Device-level lifecycle breakdown
- Production vs use phase comparison
- Finland vs France electricity grid comparison
- Fleet emissions calculator
- Sensitivity analysis

## Technologies

- Python
- pandas
- matplotlib
